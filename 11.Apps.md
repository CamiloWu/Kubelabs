# 5. Crear un recurso
New Tab (+) - Create Resource
Crear 1 recursos basandose en el archivo:
```yaml
apiVersion: apps/v1
kind: Deployment

metadata:
  name: myfrontend-deployment
  labels:
    app: mywebsite
    type: frontend

spec:
  replicas: 1

  template:
    metadata:
      name: myapp-pod
      labels:
        app: mywebsite
        type: frontend

    spec:
      containers:
        - name: nginx-container
          image: nginx

  selector:
    matchLabels:
      app: mywebsite
      type: frontend
```
