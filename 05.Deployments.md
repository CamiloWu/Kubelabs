## Crear el archivo deployment.yml

```yml
apiVersion: apps/v1
kind: Deployment

metadata:
  name: myfrontend-deployment
  labels:
    app: mywebsite
    type: frontend

spec:
  replicas: 4

  template:
    metadata:
      name: myapp-pod
      labels:
        app: mywebsite
        type: frontend

    spec:
      containers:
        - name: nginx-container
          image: nginx

  selector:
    matchLabels:
      app: mywebsite
      type: frontend
```
